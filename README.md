# AgentAnycast Website

The official homepage for [AgentAnycast](https://github.com/AgentAnycast) — a decentralized P2P runtime for the A2A (Agent-to-Agent) protocol.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is output to `dist/`.

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions on every push to `main`.

## Tech Stack

- [Vite](https://vite.dev/) + [React](https://react.dev/) + TypeScript
- [Tailwind CSS v4](https://tailwindcss.com/)
